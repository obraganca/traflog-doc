# Pátios — Listagem

Tela inicial do módulo, acessada via **Cadastros**. Exibe todos os pátios cadastrados em formato de tabela.

## Colunas da tabela

| Coluna | Descrição |
|---|---|
| Descrição | Nome do pátio |
| CEP | CEP do endereço |
| Logradouro | Rua/Avenida do endereço |
| Bairro | Bairro |
| N° | Número do endereço |
| Complemento | Complemento do endereço |
| Cidade | Cidade |
| UF | Estado |
| Ações | Ícones de ação por linha (ver abaixo) |

## Recursos da tabela

- **Filtro por coluna**: cada coluna possui um campo de filtro logo abaixo do cabeçalho.
- **Agrupamento de linhas**: área "Arraste aqui para definir os grupos de linhas" permite arrastar uma coluna para agrupar os registros por ela (ex: agrupar por Cidade).
- **Linhas**: contador no rodapé mostra a quantidade total de registros exibidos.

## Ações disponíveis

| Botão/Ícone | Função |
|---|---|
| **Novo** (canto superior direito) | Abre a tela [[Pátios - Novo Pátio]] |
| 🔍 (lupa, por linha) | Consultar/editar informações do pátio selecionado |
| 🗺️ (grade, por linha) | Abrir [[Mapa do Pátio]] daquele pátio |
